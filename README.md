# KI-Workflow für Hochschullehre

Dieses Repository enthält ein **Quarto-Book** zur strukturierten Nutzung von KI in der Hochschullehre.  
Im Fokus steht kein einzelnes Tool, sondern ein **klarer Workflow** – von der Lehrfrage bis zur interaktiven Lerneinheit.

## Worum es geht

Das Buch beschreibt einen sechsstufigen Prozess:

1. **Recherche** (Perplexity)  
2. **Grounding & Synthese** (NotebookLM)  
3. **Didaktische Transformation** (Gemini / ChatGPT)  
4. **Folienproduktion** (MARP)  
5. **Interaktivität** (H5P oder Quarto)  
6. **Video** (HeyGen)

Ziel ist es, akademische Qualität, didaktische Wirksamkeit und praktische Umsetzbarkeit miteinander zu verbinden.

## Für wen?

- Hochschullehrende  
- Lehrentwickler:innen  
- Kolleg:innen mit unterschiedlichem technischen Vorwissen  

Vorausgesetzt wird **keine** tiefgehende Technikkenntnis – Konzepte werden niedrigschwellig eingeführt.

## Technischer Rahmen

- Quarto (HTML-first)
- Markdown als Inhaltsformat
- Optional: Git/GitHub für Versionierung und Zusammenarbeit

## Mitarbeit & OER

Beiträge, Korrekturen und Erweiterungen sind willkommen.  
Das Buch ist als **Open Educational Resource (OER)** gedacht und soll gemeinsam weiterentwickelt werden.

> KI ist hier kein Ersatz für Didaktik – sondern ein Werkzeug für bessere Entscheidungen.
